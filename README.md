# Dapper - Exemplos

Exemplos focado na utilização do QueryMultiple

Tecnologias utilizadas:

* Dapper
* Framework 2.1
* ASP.NET Core
* Swagger
* Sql Server Express
* Northwind database

# Como rodar

1. Abra o projeto
2. Restaure o banco Northwind (Sql > Northwind.rar)
